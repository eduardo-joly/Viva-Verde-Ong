# 🌿 ONG Viva Verde

Um website institucional moderno, responsivo e interativo projetado para a **ONG Viva Verde**, uma organização focada na preservação ecológica, conscientização ambiental e reflorestamento.

---

## 🚀 Funcionalidades

- **Calculadora de Pegada de Carbono (CO₂)**: Ferramenta interativa desenvolvida em JavaScript para calcular a emissão estimada de toneladas de CO₂ baseada no estilo de vida do usuário, sugerindo a quantidade ideal de árvores nativas a serem plantadas para compensar seu impacto ambiental.
- **Formulário de Inscrição de Voluntários**: Área dedicada a captar novos colaboradores com validação e feedback dinâmicos ao usuário.
- **Design Moderno Ecológico**: Interface com paleta de cores verde-floresta elegante baseada em Glassmorphism e efeito blur no plano de fundo.
- **Totalmente Responsivo**: Layout estruturado com Flexbox e Grid CSS, garantindo uma navegação suave em qualquer tamanho de tela (mobile-first).

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação de conteúdo semântico para SEO e acessibilidade.
- **CSS3**: Estilos customizados, gradientes ecológicos modernos, transições suaves de hover e design responsivo fluido.
- **JavaScript (ES6+)**: Lógica de cálculo da pegada de carbono, manipulação do DOM e envio interativo do formulário.

---

## 📂 Estrutura do Projeto

```text
├── index.html     # Página principal contendo as seções e o formulário
├── style.css      # Folha de estilo visual com design de glassmorphism
└── main.js        # Lógica de programação da calculadora e do voluntariado
```
